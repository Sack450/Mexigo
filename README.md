# MexiGo
Welcome to MexiGo! This project is a digital solution designed to boost consumption in micro and nano businesses while also promoting tourism in Mexico.

## 🚀 Description
MexiGo is a platform that integrates Firebase services and geolocation to offer a seamless experience. The project is currently in the development and infrastructure optimization phase.

## 🛠️ Technologies Used
Backend/Database: Firebase (currently configured in Test Mode).

Languages: JavaScript, TypeScript.

Infrastructure: Integration with Google services, with a view to migrating to Google AI Studio/Antigravity by 2027.

## ⚠️ Project Status
Status: In development / Under maintenance.

### Security Note: Currently, client access to the database is restricted due to Firebase security rules. These rules need to be updated to restore full functionality.
